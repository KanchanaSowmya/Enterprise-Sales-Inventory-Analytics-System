# 📊 Enterprise Sales Inventory Analytics System
## 📌 Overview

The **Enterprise Sales Inventory Analytics System** is a comprehensive SQL Server database project designed to simulate real-world business operations. It demonstrates enterprise-level database design principles including normalization, relational modeling, stored procedures, auditing, and analytics.

The system manages inventory, processes sales transactions, tracks stock levels, and provides business insights through structured queries and database objects.

---

## 🚀 Features
- Fully normalized relational database design (3NF)
- Inventory management and stock tracking system
- Sales order processing and transaction handling
- Stored procedures for business logic automation
- Audit tracking for data changes and history
- Business analytics and reporting queries
- Complete sample data for all tables using Insert scripts
- Over 30 relational database objects (tables, procedures, views, etc.)

## 🛠️ Technologies Used
- Microsoft SQL Server
- T-SQL (Transact-SQL)
- Stored Procedures
- Database Normalization
- Relational Database Design

## 🗂️ Project Structure
- `Enterprise Sales Inventory Analytics System.sql`   → Database schema creation
- `Insert.sql`                                        → Sample data insertion
- `StoredProc.sql`                                    → Stored procedures logic

## 🧠 Key Concepts Implemented
- Entity Relationship Design (ERD)
- Primary and Foreign Keys
- Joins and Relationships
- Stored Procedures
- Data Integrity Constraints
- Audit Logging System
- Business Reporting Queries

## 📊 Sample Use Cases
- Track product inventory levels
- Manage customer sales orders
- Generate sales performance reports
- Monitor stock movements
- Maintain audit history of changes

## ⚙️ How to Run
1.Open Microsoft SQL Server Management Studio (SSMS)
2.Run Enterprise Sales Inventory Analytics System.sql to create database schema
3.Execute Insert.sql to populate sample data
4.Run StoredProc.sql to create stored procedures
5.Execute sample queries for reporting
